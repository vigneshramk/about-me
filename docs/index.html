Research & Projects

Vision based Synthesis and Deployment of Robotic Vehicles using Symbolic Controllers

Hybrid Control Systems group, TU Munich, Germany | DAAD WISE Fellowship| May 2016- Aug 2016 | Supervisor: Prof. Dr. Majid Zamani

A group of robots in an urban environment, with predefined target sets and LTL specifications. These specifications include reach-and-stay, recurrently visiting two targets and sequential visit of targets spread across the platform. These bots implement the symbolic controller, based on SCOTS, a software tool built by HCS group in TU Munich for the automatic controller synthesis for nonlinear control systems based on symbolic models. Synthesized a symbolic controller based on SCOTS for such complex LTL specifications. The system uses vision to find the kinematics of the moving robots in real-time and establishes the closed loop control by communicating to the server through UDP. Designed a framework for acquiring real-time kinematics of robot from live feedback given by overhead camera and sending over these co-ordinates to the robot as UDP packets. Implemented the sparse-matrix based Symbolic Controller as C++ program in the robot’s Linux console hence satisfying the complex LTL properties as shown in the video.

Vision based Vehicle Trajectory Prediction using an Omnidirectional Camera

Computer Vision Lab, Indian Institute of Technology(IIT), Kanpur | May 2015 - July 2015 | Supervisor: Prof. Dr. K. S. Venkatesh | Presented at ICACCI 2016

A practical method is presented to predict the future spatial-temporal trajectories of multiple vehicles at road intersections in real time using a catadioptric omnidirectional camera equipped with an Equiangular mirror. Implemented an object tracking algorithm in a traffic scene using the integrated CAMShift-Kalman Filter algorithm for improved occlusion-handling. Domain transformation of the tracked objects location and velocity from image space to real world is done using a geometrical model. Formulated a motion model to predict the trajectory of the vehicles in the scene for the next few seconds.

ECG Signal Analysis using Wavelet Transform, S-Transform, Recurrence Quantification and Hjorth Parameter Estimation for Classification of Cardiovascular diseases

Spider R&D, NIT Trichy | Researcher | Presented at ICACCI 2016 | Mar 2016 - Sep 2016

The spontaneous classification of cardiovascular diseases is a challenging task and can be made more feasible with proper ECG fluctuation analysis. We perform a qualitative analysis of the ECG data using complex Gaussian wavelets to investigate the multi-scale, self similar behaviour and deviation via phase plots of the wavelet cross spectrum of the ECG signals. We analyze ECG signals using S transform to overcome the limitations of continuous wavelet transform and make the results more consistent and reliable. Recurrence quantification analysis (RQA) is a nonlinear analysis tool for investigating the non- stationarity of dynamical systems through the recurrence number and duration in the phase space trajectory of signals. The phase transitions of perturbed ECG signals from normal ECG due to cardiac abnormalities are captured by RQA, while the Hjorth parameters provide the time domain statistical information about the signal. The adequacy of the experimental results validates the efficacy for a integrated Computer Aided Diagnostic (CAD) system for diagnosing different cardiovascular diseases using the above methods.

Digit Recognition for Tamil Language using Kaldi ASR

Speech Processing Lab, Indian Institute of Technology (IIT), Madras | Dec 2015 - Jan 2016| Supervisor: Prof. Dr. S. Umesh

And end-to-end application for digit recognition for the language ‘Tamil’ from continuous speech input recorded and decoded in real time. My primary task was to formulate the acoustic and language models for the language ’Tamil’. The speech recognizer was built using Kaldi ASR Toolkit. I developed a system using C++ to build,train and test the recognizer based Hidden Markov Models using Kaldi ASR. The digit recognizer is backed up by a front end real-time decoding user interactive application using PHP and Asterisk. The system was made robust to all possible varied utterances of each digit. Achieved a WER (Word Error Rate) 23.48 with the optimised vocabulary making a trade-off between robustness of the system and improving accuracy.
